What is Python language?
Python is a widely used high-level, general-purpose, interpreted, dynamic programming language.
Its design philosophy emphasizes code readability, and its syntax allows programmers to express concepts
Python supports multiple programming paradigms, including object-oriented, imperative and functional programming 
It features a dynamic type system and automatic memory management and has a large and comprehensive library
